# webspatial-ai

MCP server for debugging and testing [WebSpatial](https://webspatial.dev/docs) apps on **PICO OS 6** (Android-based Swan emulator, via ADB) and **visionOS**.

## Capabilities

- Build, deploy, and exercise WebSpatial flows with an AI agent
- Observe behavior in the emulator or device and iterate on code from what it sees

## Stretch

- Live preview of generated WebSpatial code on the emulator (WebSpatial playground)
