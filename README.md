# Overview
Build an AI to automatically build, deploy and test WebSpatial applications that can easily be opened on the emulator or device.

# Goal
- Create a simple WebSpatial app using AI prompt
- Automatically run the WebSpatial app using AI
- Test the WebSpatial app in the emulator or device using AI
- AI should make corrective code updates based on observations in the emulator
## Stretch
- Live preview generated WebSpatial code on the emulator, aka WebSpatial playground
